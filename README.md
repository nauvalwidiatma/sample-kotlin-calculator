# Simple Kotlin Calculator

Sample project Android Kotlin sederhana untuk tes upload ZIP di builder APK.

## Fitur

- Input angka 0-9
- Operasi `+`, `-`, `x`, `/`
- Tombol `C`
- Tombol `=`

## Catatan

- Ini project Android Kotlin sederhana dengan XML layout
- Belum termasuk Gradle wrapper (`gradlew`)
- Cocok untuk sample struktur project dan dasar build pipeline
